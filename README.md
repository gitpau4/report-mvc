![Screenshot of the website](public/img/readme_screenshot.png)

## Prerequisites
You have PHP, Composer and npm.

## Get started
```
git clone git@github.com:gitpau4/report-mvc.git
cd report-mvc
composer install
npm install
npm run build
symfony server:start
```
